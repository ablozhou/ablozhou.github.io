#  					 				

				

 					  					  					

[周海汉](http://blog.csdn.net/ablo_zhou) /文

2010.1.28

 

公司一台服务器，装的windows 2003, 远程桌面连上去，鼠标很凝滞，一顿一顿的。因为是局域网连接，不存在网络或带宽不够的问题。同一个机架的别的机器都很顺畅。内存2G，占用了1G。CPU占用率很低。网卡流量几乎为0。百思不得其解。

 

后面有同事反复探索，偶然将其换了一个主题为windows 经典，结果豁然开朗，再没有操作不畅的现象。

 

主题也会影响操作，我做梦都没想到。
