html提供标签<sup>和<sub>来写上标和下标。下面是一些示例

源代码：

    
    x<sup>2</sup>+x+1=1;
    WWW<sup>[1]</sup>
    H<sub>2</sub>O


浏览器显示结果：

```


x2+x+1=1;
WWW[1]
H2O
```

这在html和xhtml都支持，各浏览器也都支持。

感觉直接用标签还是挺方便的。
